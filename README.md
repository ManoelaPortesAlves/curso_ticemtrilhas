# 🚀 React - TIC em Trilhas

Repositório criado para armazenar os estudos, exercícios e projetos desenvolvidos durante o curso de **React** do programa **TIC em Trilhas**.

O objetivo deste repositório é acompanhar a evolução no aprendizado do ecossistema React, aplicando boas práticas de desenvolvimento front-end, organização de código e construção de interfaces modernas.

---

## 📚 Sobre o curso

O curso aborda conceitos fundamentais e avançados do React, incluindo:

- Fundamentos do React
- Componentização
- Hooks
- Gerenciamento de estado
- Consumo de APIs
- Boas práticas de desenvolvimento
- Organização de projetos
- Performance e reutilização de componentes

---

## 🛠️ Tecnologias utilizadas

- React
- TypeScript
- Vue / Next.js (quando aplicável)
- HTML5
- CSS3
- JavaScript (ES6+)
